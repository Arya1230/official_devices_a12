# Installation Guide For PixelPlusUI on Begonia/Begoniain

### CLEAN FLASH (RECOMMENDED) :
1. Flash ROM (and Magisk if needed)
2. Format Data
3. Reboot

### DIRTY FLASH IF AN UPDATE
1. Clear Dalvik and Cache in wipe
2. Flash ROM (and Magisk if needed)
3. Reboot
