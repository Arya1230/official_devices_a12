# Installation Guide For PixelPlusUI on Redmi 6 Pro (sakura)

### CLEAN FLASH (RECOMMENDED)
Download rom 
Reboot recovery
Format data (if u coming from miui rom  )
Wipe all except internal and ext sd card or otg
Flash rom 
Reboot

### DIRTY FLASH
Download rom 
Reboot recovery
Flash rom 
Reboot

## Notes: 
- Clean flash if you face any bugs
