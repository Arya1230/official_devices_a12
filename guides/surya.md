# Installation Guide For PixelPlusUI on surya/karna OSS Based

### CLEAN FLASH :
- Download the latest build
- Reboot to recovery.
- Take a backup.
- Format Data
- Flash the latest build
- Reboot :D

### DIRTY FLASH :
- Download the latest build
- Reboot to recovery
- Flash the latest build
- Wipe Cache
- Reboot :D
